Aplicación Java utilizando Lógica Difusa

Esta aplicación utiliza lógica difusa para controlar la velocidad de un ventilador según la temperatura ambiente. En lugar de tomar decisiones rígidas, como encender o apagar, evalúa qué tan "fría", "templada" o "caliente" está la temperatura usando conjuntos difusos, y con base en eso determina una velocidad adecuada del ventilador (baja, media o alta). Por ejemplo, si la temperatura es muy baja, el ventilador girará lentamente; si es muy alta, funcionará a máxima velocidad. Este enfoque permite transiciones suaves y respuestas más naturales frente a los cambios de temperatura.
